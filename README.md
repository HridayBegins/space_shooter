 Space Shooter 🚀

A retro arcade space shooter game built with Python and Pygame.

Gameplay
Survive waves of enemies and get the highest score!

Controls
| Key | Action |
|-----|--------|
| W / A / S / D | Move |
| Spacebar | Shoot |

  How to Run

  Option 1 – Run from source
1. Install Python
2. Install dependency:
pip install pygame
3.pip install pyinstaller
4. Run the game:
python main.py

  Built With
- Python
- Pygame

  Features
- Collision detection
- Scoring system
- Sprite animations
- Retro arcade style
